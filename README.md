# Notebook

## Description
A command-line notebook that has a menu to interact with notes. The menu is
the following:
1) Show all Notes
2) Search Notes
3) Add Note
4) Modify Note
5) Quit

## Modules

contains Note and Notebook classes.

[menu.py](https://github.com/linvieson/notebook/blob/main/menu.py) -
contains Menu class.


## Results
The result of running a program is a command-line notebook. It displays a menu
to a user, which has 5 options: to show all notes, to search notes, to add a
note, to modify a note and to quit a program.
